Task 2: CSS Positioning Examples

This task showcases three types of CSS positioning:

  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to its nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport.

This task demonstrates how to make an element responsive using media queries:

  • On small screens, the element takes up 50% of the width.
  • On larger screens, it adjusts to 25% width.
