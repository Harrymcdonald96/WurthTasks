Task 1: Difference Between Div and Section

This task explains the difference between a div and a section:

  • Div: A generic container for grouping elements without providing any semantic meaning.
  • Section: A semantic container used to group related content with meaningful structure.

Here is an example that demonstrates the difference:


          
      
      

        
Blog Title

        
This is a blog post about web development.

      

      
      
      

        

          
Blog Title

        

        
This is a blog post about web development.

Explanation:

  • The first example uses generic div elements to group content. It is functional but does not provide additional meaning.
  • The second example uses a section element to group related content semantically, improving accessibility and SEO.

Task 3: Difference Between Ordered and Unordered Lists

This task explains the difference between ordered and unordered lists:

  • Ordered List: Creates a list with numbered items.
  • Unordered List: Creates a list with bullet points.

Ordered List Example:
1. First item
2. Second item
3. Third item

Unordered List Example:
- Item one
- Item two
- Item three
